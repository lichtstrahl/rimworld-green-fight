# Обучение моддингу Rimworld

### Установка

1. Собрать проект, получив `GreenFight.dll` на выходе.
2. Создать папку под мод в `<Rimworld>/Mods/GreenFight`.
3. Перенести файл из п.1 в `Assemblies` в модификации.
4. Перенести файлы из папки `RimworldData` данного репозитория в подификацию. См. соответствующий `README.md`
